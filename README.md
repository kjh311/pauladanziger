Creating a website for Paula Danziger, showcasing her books and life.

* Flexbox
* Photoshop
* Progressive Image Enhancement (https://github.com/Wildtyto/Interlace.js)

* SASS
* Jquery
* Smooth Scrolling on Anchor Tags
* Responsive Design
* CSS Animations
* Modal lightbox Image Carousels
